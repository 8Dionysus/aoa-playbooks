# AOP-Q-AGON-0003: Mechanical Trial Review

## Intent

Review whether each seeded mechanical trial remains compact enough to be a playbook rather than a giant orchestration script.

## Done when

- Each trial has a clear trigger boundary.
- Each trial names expected evidence and fallback posture.
- Each trial keeps verdicts, scars, retention, rank, runtime, and ToS promotion outside `aoa-playbooks`.
