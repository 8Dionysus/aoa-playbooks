# Agon Trial Playbooks

## Purpose

Wave VI lands the first **pre-protocol mechanical trial playbooks** for Agon.

A trial playbook is a recurring scenario composition. It coordinates actors, assistant service support, lawful moves, expected evidence, owner review, fallback posture, and stop-lines.

It does not create an arena session.

## Core invariant

```text
The playbook rehearses the test. The arena does not yet exist.
```

## Why this belongs here

The trial routes span more than one bounded workflow and need recurring choreography. That is playbook-layer meaning.

The underlying move law remains center-owned. Execution workflows remain skill-owned. Proof checks remain eval-owned. Durable scars and retention remain memo-owned.
