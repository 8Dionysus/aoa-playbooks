# Titan Service Cohort Playbook

## Opening

Summon Atlas, Sentinel, and Mneme explicitly.

```text
Atlas: map the route and owner boundaries.
Sentinel: name drift risks and gates.
Mneme: name provenance posture and receipt candidates.
```

## During work

Use Atlas before structure changes. Use Sentinel before accepting risk. Use Mneme before memory or lineage claims.

## Forge gate

Forge appears only when mutation is explicit.

```text
Intent
Targets
Validation
Rollback or stop
```

## Delta gate

Delta appears only when judgment is explicit.

```text
Question
Evidence
Criterion
Verdict form
```

## Closeout

Every Titan-backed session should close with:

```text
active roster
conditional roster status
route frame
risk gates
memory posture
mutation gate status
judgment gate status
validation result
next seed or closure note
```
