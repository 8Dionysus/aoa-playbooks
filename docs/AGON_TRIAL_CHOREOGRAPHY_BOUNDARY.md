# Agon Trial Choreography Boundary

`aoa-playbooks` owns the choreography of a recurring trial route.

It may name when the route starts, which role homes participate, which assistant support forms may package receipts, which lawful moves are relevant, which future owner handoffs may be requested, and which evidence should exist when the route ends.

It may not own:

- no_arena_session_creation
- no_sealed_commit_runtime
- no_live_move_execution
- no_verdict_authority
- no_scar_write
- no_retention_schedule
- no_rank_mutation
- no_tos_promotion
- no_hidden_assistant_contestant
- no_runtime_substrate_claim

## Boundary phrase

```text
Trial choreography is not arena sovereignty.
```
