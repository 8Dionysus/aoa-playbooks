# Titan Memory Loom Playbook

## Entry

Start after a Titan session, console closeout, app-server bridge replay, or planting report.

## Flow

1. Mneme ingests candidate records.
2. Atlas maps record to owner surfaces.
3. Sentinel checks authority drift.
4. Operator asks recall queries.
5. Delta appears only for judgment.
6. Forge appears only for approved memory-surface mutation.

## Exit

Create a digest and list writeback candidates. Do not auto-write personal or project memory.
